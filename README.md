## Project Overview

This repository contains my Digital Literacy Portfolio created as part of the CSE0001 course at VIT Bhopal.

The goal of this project is to understand and apply important digital skills that are useful for students today. Through this project, I explored how to build a professional online presence, use different digital platforms, communicate properly online, and stay safe from cyber threats.

The project is divided into five tasks, each focusing on a different aspect of digital literacy.

## Summary of Tasks

### Task 1: Digital Literacy Infographic

In this task, I created an infographic to explain the meaning of digital literacy, safe internet practices, and the importance of maintaining a professional online presence.

### Task 2: Digital Portfolio

I created and updated my profiles on platforms like GitHub and LinkedIn to start building my digital presence as a student.

### Task 3: Platforms

I explored a coding platform by solving a beginner-level problem and also created a Google Form quiz to test digital literacy awareness.

### Task 4: Email Etiquette

I wrote two professional emails and created a checklist of do’s and don’ts for responsible social media use.

### Task 5: Cybercrime Awareness

I prepared a case study on cybercrime and created a checklist to help students stay safe online.
